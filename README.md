## getPPTWords
### 一个纯前端的PPT文字提取器

## 在线体验地址

## 背景
网上成熟的PPT转Word文档工具有很多，不过难点在于区分哪些是必要的图，哪些是不必要的
本来想接入例如火山引擎的doubaoAPI调用视觉理解能力，但是我暂时没精力处理例如下方的几个问题：
- 如何返回图片？
- 在服务端如何处理PPT可以使用最少的token？
- 其他的相关细节问题
所以我想干脆只提取文字好了，图片我自己弄
网上就没有提取PPT文字工具了吗？
实际上有许多，但是我发现一个致命的问题：
- 无法提取SmartArt中的文字信息
所以我想了想还是自己做一个吧

## 技术选型
作为一个简单的脚本，我一开始准备使用的是Python语言，结合python-pptx库提取文字
但是我实际上是第一次接触这个库，我找不到如何提取SmartArt中的文字，网上和AI也没有给我一个很好的解决方案
而且我希望它有一个直观的界面可以让我使用，所以为了方便考虑我最终选择了node.js

我利用VibeCoding,使用了jszip库，直接将PPT拆解，正则匹配到xml的相关内容后输出
原理就这些，毕竟只是个小脚本，感兴趣的可以查看源码
